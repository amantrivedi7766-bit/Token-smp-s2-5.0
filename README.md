# TokenSMP 5.0

TokenSMP is a Spigot/Paper plugin skeleton for a token-based SMP server. The repository contains the full requested Maven layout, GitHub Actions build workflow, plugin resources, commands, managers, listeners, GUI classes, token models, and ability placeholders.

## Build

```bash
mvn package
```

The generated plugin jar is written to `target/token-smp-5.0.0.jar`.
